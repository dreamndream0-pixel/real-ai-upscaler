# Premium Upscaler 原本風格雙模式版

這版保留原本白色高級 UI / Remini 背景 / 漸層背景 / 滑桿對比，並加入 Natural / Enhance 雙模式。

## 本機啟動
```bash
npm install
touch .env
```

在 `.env` 裡放：
```env
REPLICATE_API_TOKEN=你的token
PORT=3000
```

啟動：
```bash
npm start
```

打開：
http://localhost:3000

## Render
